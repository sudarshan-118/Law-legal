## 📌 Overview

**Law & Legal** is an AI-powered web platform built to make legal guidance more accessible and user-friendly. The application helps users interact with legal information through a clean interface, intelligent assistance, and modern web technologies.

The platform focuses on simplifying complex legal processes and improving awareness by providing users with a smarter and more intuitive digital legal experience.

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/sudarshan-118/Law-legal.git
```

### 2️⃣ Navigate into the Project

```bash
cd Law-legal
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Run the Development Server

```bash
npm run dev
```

Now open:

```bash
http://localhost:3000
```
